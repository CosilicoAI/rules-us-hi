# rules-us-hi

Hawaii state tax and benefit rules encoded in Cosilico DSL.

## Structure

```
statutes/       # Hawaii Revised Statutes (HRS) encoded rules
regulations/    # Hawaii Administrative Rules (HAR) encoded rules
```

## Sources

- [Hawaii Revised Statutes](https://www.capitol.hawaii.gov/docs/HRS.htm)
- [Hawaii Administrative Rules](https://ltgov.hawaii.gov/the-office/administrative-rules/)
- [Hawaii Department of Taxation](https://tax.hawaii.gov/)
